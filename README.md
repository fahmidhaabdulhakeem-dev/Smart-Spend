# 💰 SmartSpend — Premium Budget Tracker

SmartSpend is a high-fidelity,  personal finance dashboard built using pure HTML, Vanilla CSS, and Vanilla JavaScript. It offers a clean glassmorphic interface, real-time analytics visualizations, custom themes, active budget limits, and quick import/export capabilities for modern wealth management.


---

## ✨ Features

- **📊 Visual Data Analytics**: Interactive, real-time Doughnut chart powered by Chart.js representing categories breakdown.
- **📈 Live Balance Calculation**: Instant balance adjustments highlighting monthly surplus (green) or deficit (red).
- **➕ Full CRUD Support**: Seamlessly add, view, edit, search, and delete both Income and Expense transactions.
- **🎨 4 Curated Premium Themes**:
  - **Aurora Dark** (Default): Cyber-fusion dark gradient.
  - **Nordic Light**: Minimalist daylight layout.
  - **Cyberpunk Neon**: Neon pink and cyan glow.
  - **Emerald Forest**: Rich natural emerald and gold tones.
- **🎯 Active Monthly Goals**: Configurable budget limits with a color-coded visual indicator bar that updates dynamically.
- **💾 Local Storage Persistence**: Auto-saves your finance records so they persist across page refreshes and browser restarts.
- **📂 Export & Import**:
  - **Export to CSV**: Perfect for spreadsheets and recordkeeping.
  - **Export to JSON**: Creates offline database backups.
  - **Import JSON**: Restore your files back in one click.

---

## 🛠️ Built With

- **HTML5**: Structured semantic markup.
- **Vanilla CSS**: Premium layouts utilizing CSS grids, custom properties, glassmorphism filters, and CSS keyframe animations.
- **Vanilla JavaScript**: DOM Manipulation, LocalStorage management, and event pipelines.
- **Chart.js CDN**: Real-time rendering of transaction ratios.
- **Lucide Icons**: Crisp vector icons.
- **Plus Jakarta Sans (Google Fonts)**: Custom modern typography.
